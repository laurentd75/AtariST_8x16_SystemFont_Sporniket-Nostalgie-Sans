# Sporniket Nostalgie Sans

**Sporniket Nostalgie Sans** is a collection of fonts inspired by the Atari ST system : its system fonts ('high resolution' 8×16, 'low resolution' 8×8, 'tiny' 6×6), mouse pointers (standard, bee, ...) and default icons (drive icons, file and folders icons). The motto of these fonts : «faithful on screen, beautiful on paper».

## Roadmap

The collection will provide the following fonts, mostly in this order of making :

* _DONE_ - a 'console' variant, made after the 8×16 font (_ST High_) ;
* _PLANNED_ - a 'console small' variant, made after the 8×8 font (_ST Low_) ;
* _PLANNED_ - a 'console tiny' variant, made after the 6×6 font ;
* _MAYBE_ - an 'icon' variant
* _MAYBE_ - a 'sans-serif' with variable width variant, made from the 'console' variant in the spirit of the `v1` version of this project.

## Features

* At the very least, the US-ASCII range, as well as a good part of Latin-1 supplement. In other words, can be used in a terminal and text editor for programming.
* Some supplement (diacritics, ligatures, currency EURO €)
* Eventually, 
  * some special glyphes found in the original ST fonts, like the famous "R. 'Bob' Dobbs" grinning face, the desktop windows button like maximize button, resize button, scrolling arrows, etc...
  * some mouse pointers, like the busy bee, the hand, etc...
  * and the desktop icons, like the cabinet, the trash, etc...

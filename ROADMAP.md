# Roadmap

> Subject to change. The time to release is undefined

## Inspired by the Atari ST

The collection will provide the following fonts, mostly in this order of making :

* _DONE_ - a 'console' variant, made after the 8×16 font (_ST High_) ;
* _TODO_ - a 'console small' variant, made after the 8×8 font (_ST Low_) ;
* _TODO_ - a 'console tiny' variant, made after the 6×6 font ;
* _MAYBE_ - an 'icon' variant
* _MAYBE_ - a 'faux console', a variable width font made by using the glyphes from the 'console' variant.

### Features

* At the very least, the US-ASCII range, as well as a good part of Latin-1 supplement. The 'console...' variant can be used in a terminal and text editor for programming.
* Some supplement (diacritics, ligatures, currency EURO €)
* Eventually, 
  * some special glyphes found in the original ST fonts, like the famous "R. 'Bob' Dobbs" grinning face, the desktop windows button like maximize button, resize button, scrolling arrows, etc...
  * some mouse pointers, like the busy bee, the hand, etc...
  * and the desktop icons, like the cabinet, the trash, etc...
